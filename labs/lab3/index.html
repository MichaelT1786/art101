Lab 3 - File Structures and File Transfers

Challenge

This lab challenges us to learn about how files are stored and structured locally, how to add and position an image, and how to use html's list feature.

Problems

The main problem faced was manipulating the image and other elements on the screen to work in a way that looked pretty. Since "float" lets the image spill out of its container, I went back and forth trying to make the container larger or hold the image properly, ultimately resorting to manipulating the image's size to fit.

I am also currently having some problems with the image I'm using for the background not having its name updated in github, but that's a bigger issue than what I can really do here.

Reflection
